# Project Name

Wiki-clone app in React/Redux

## Built Using
* React
* React Router
* Redux
* Webpack
* Node Sass
* Node - 6.0 only

## Server-side Implementation
* Express
* Body Parser
* Router
* Nodemon

## Installation (client-side)

1. Run `yarn install`
2. Run `npm rebuild node-sass`
3. Run `yarn start`

## Installation (slient-side)

1. Run `yarn install`
2. Run `yarn server`
